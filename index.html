Multi Agent System: Retriever + ChartGenerator

View on GitHub

This project is a multi-agent system built using LangGraph in combination with Haystack, where:

Retriever Agent: Uses a Haystack RAG pipeline as a tool to fetch relevant information from user-uploaded documents.

Chart Generator Agent: Uses a Python REPL tool to generate charts based on retrieved information.

Retrieval Augmented Generation (RAG) Agent

View on GitHub

This project implements a Retrieval-Augmented Generation (RAG) system that enhances generative AI models with information retrieval capabilities. It allows users to upload documents, query the system, and receive responses based on both stored knowledge and real-time retrieval.

RMSE-Based Assessment of Model Complexity and Overfitting in Polynomial Regression Frameworks

View on GitHub

This project demonstrates overfitting in machine learning using polynomial regression on the Advertising dataset. By progressively increasing the degree of polynomial features, we analyze the bias-variance trade-off and how model complexity impacts training and testing performance.

Data Analysis: Webpage Metrics Animemangatoon

View on GitHub

This project analyzes user engagement data for various articles on AnimeMangaToon using A/B testing and various data visualizations. The objective is to gain insights into user behavior and suggest improvements to increase user retention and reduce bounce rates.

Data Analysis: Ambient Air Quality of Maharashtra 2015

View on GitHub

This project analyzes air quality data from Maharashtra for the year 2015, focusing on three key pollutants: SO2, NO2, and RSPM/PM10. The analysis aims to assess pollutant levels, identify trends, and provide insights into potential pollution hotspots and implications for public health.

Performance Analysis: Solar Plants in India

View on GitHub

This project analyzes solar power generation data from two Indian solar plants, comparing power output trends, inverter efficiency, and irradiation levels to optimize energy utilization and detect underperforming inverters.

ISBN 13&10 Checksum Calculator

View on GitHub

The ISBN 13&10 Checksum Tool is a Python-based utility for calculating the check digits for both ISBN-10 and ISBN-13 numbers. \ This tool helps users verify the validity of ISBN numbers by generating the correct check digit based on the initial part of the ISBN.
